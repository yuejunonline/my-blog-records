其他步骤不变 恢复设置 连接wifi 第一步：安装“DNS证书文件” 1. 进入appletv -> 设置 -> 通用 2. 把光标移动到“将数据发送给Apple”这一项上，什么都不操作 3. 然后按一下播放/暂停键（右下角的一个按键），就出现“添加描述文件”，选择“好” 4. 点“清除”，然后输入 tiancaoyezi.com/fix 提交，看到出现了appletv-fix这一项，就说明DNS证书安装成功了！（如出现报错，请检查网络连接正常，也可尝试这个地址 \*\*/zRLJ6Js（区分大小写） 。还不行需进入设置 -> 通用 -> 还原 -> 还原所有设置，重新加入网络再安装。） 第二步：设置“iTunes Store的位置”和“DNS服务器地址” 5. 进入设置 -> iTunes Store -> 位置 ->United States美国，然后按Menu键退出到桌面，会看到有很多图标出来。 6. 再进入设置 -> 通用 -> 网络 -> WiFi(进入) -> 自已的WiFi网络名称(进入) -> 配置DNS改为手动 -> 改为115.029.189.118，完成 7. 最后按Menu键退出到桌面，进入“预告片” 这个程序图标，就可看国内的视频网站了。   最后感谢 xi811103 同学提供解决的DNS ！
